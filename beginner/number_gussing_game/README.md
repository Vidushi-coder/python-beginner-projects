# Number Guessing Game (Python)

This is a Python-based **CLI Number Guessing Game** where the computer randomly selects a number and the user tries to guess it correctly.

The game provides hints such as **Too High**, **Too Low**, or **Very Close**, counts the number of attempts, and allows the user to **play multiple rounds**.
